description: Tour Diary is an application I've been wanting to build for such a long time. When musicians aren't playing a gig, they're usually talking about other shows that are going to happen or talk about what has already happened. I finally bit the bullet and set up my very own tour diary. The catalogs the shows that I've played over the years, the flyers, and other information about each gig. This was built with a react vite template which is hosted on Netlify and is using Pocketbase as a database hosted on my own Unraid server using Docker! There are many ways I can improve this one but I'm glad to finally have an MVP.
